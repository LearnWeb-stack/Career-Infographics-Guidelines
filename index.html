A streamlined guide for high school students

üìã Before You Start

  • Define your main message - What's the ONE key takeaway?
  • Know your audience - Who will see this infographic?
  • Gather all content - Stats, facts, quotes from the articles
  • Sketch a rough layout - Hand-drawn wireframe

üé® Design Basics

  • Choose 3-5 colors max - Use color psychology (blue = trust, green = growth)
  • Pick 2-3 fonts - One bold header, one readable body text
  • Create visual hierarchy - Biggest = main title, smallest = details
  • Use consistent icon style - All outlined OR all filled, same size throughout

üìä Content Organization

  • Follow reading flow - Top to bottom, left to right
  • 5-second rule - Main message clear immediately
  • Break up text - No paragraphs longer than 3 lines
  • Include data sources - Small attribution at bottom

üîß Technical Setup

  • Choose your tool - Canva (beginner), Figma (intermediate)
  • Set dimensions - 1080x1080px (Instagram) or 8.5x11" (print)
  • Save multiple formats - Working file + PNG/PDF for sharing

‚úÖ Final Check

  • Proofread everything - Spelling, grammar, numbers
  • Test readability - View at actual size, ask someone to review
  • Optimize file size - Compress without losing quality
  • Make it shareable - Right size for intended platform

üí° Success Tips

  • Start simple - Better to do fewer things well
  • Use white space - Don't fill every inch
  • Stay consistent - Same spacing and alignment throughout
  • Test on mobile - Make sure it's readable on phones

üéØ Focus on These 4 Key Elements from Your Articles:

Article 1 - Front-End: Vocabulary (HTML, CSS, JavaScript), 16% job growth, responsive design process, AI integration skills
Article 2 - Back-End: Server-side languages (Python, Java), APIs & databases, 16-20% growth, cloud platforms (AWS, Azure)
Article 3 - UI/UX: Design Thinking process (5 steps), 13-18% growth, accessibility focus, AI tool integration
Article 4 - DevOps/QA: CI/CD pipelines, testing methods, 15-21% growth, AI model testing skills

üìä Basic Rubric - Information Accuracy Check:

  • Job growth statistics correct - Front-End & Back-End: 16%, UI/UX: 13-18%, DevOps/QA: 15-21%
  • Key technologies accurate - HTML/CSS/JS, Python/Java/Node.js, Figma/Sketch, CI/CD
  • Role definitions clear - What each tech role actually does day-to-day
  • AI integration mentioned - How AI impacts each role and future skills needed
  • Industry examples included - Healthcare, fintech, e-commerce, AI startups
